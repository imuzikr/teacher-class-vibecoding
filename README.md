# Vibe Coding Starter

비개발자와 초보자를 위한 8차시 바이브 코딩 학습 웹앱입니다.

## 화면 구조

- 랜딩 페이지
- 나의 강의실
- 1차시부터 8차시까지의 차시 상세 페이지

## 설계 방향

- 어려운 개발 용어를 쉬운 말로 다시 설명
- 매 차시마다 바로 써볼 수 있는 AI 프롬프트 제공
- 차시별 결과물, 체크리스트, 코치 메모 중심의 실습형 구성
- 로컬 저장소 기반 학습 진도 저장

## 실행 방법

```bash
npm install
npm run dev
```

## 참고

- 진도는 브라우저 `localStorage`에 저장됩니다.
- 학습 흐름은 8차시 기준으로 구성되어 있습니다.
